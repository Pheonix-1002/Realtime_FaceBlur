# Real-time Face Blur using OpenCV DNN

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python) 
![OpenCV](https://img.shields.io/badge/OpenCV-4.x-brightgreen) 

Real-time face detection and blurring using OpenCV’s Deep Neural Network (DNN) module. This tool helps anonymize faces in live webcam feeds or video streams.

## Features

- Real-time face detection using a pre-trained DNN (`res10_300x300_ssd_iter_140000.caffemodel`)
- Automatic Gaussian blur applied to detected faces
- Displays “No Face Detected!” if no faces are found
- Works directly with webcam input

## Technologies Used

- Python 3.x
- OpenCV (`cv2`)
- Pre-trained Caffe DNN model
